# 中文工程文献检索题目集（engineering 版）

用途：用于测试 `search_mode: engineering` 下的中文工程与应用技术主题到英文检索式生成、检索源选择、相关性筛选和后续文献分析流程。题目覆盖机械、电气、土木、材料、能源、制造、交通、航空航天、控制、机器人硬件、化工过程和工业安全等场景。

推荐默认设置：
- `search_mode`: `engineering`
- 推荐检索源：Crossref、OpenAlex、Semantic Scholar、CNKI；控制、机器人、优化、信号处理、计算工程和模型密集型题目可加 arXiv；生物医学工程、临床设备、职业安全/环境健康相关题目可加 PubMed。
- 中文题目若选择 CNKI，应保留中文原始 query；英文源应使用英文工程术语和常用缩写。

标注位置：本文件只保存“检索题目”。每次实际检索后，候选文献会自动追加到 `检索标注记录.md`，请在那里填写 `人工判断`、`错误类型` 和 `备注`。

常用错误类型：
- `query_too_broad`：检索式泛化到大工程主题，如把热失控预测泛化为 battery management。
- `query_too_narrow`：检索式过窄，漏掉常见缩写、同义词或工程条件。
- `missing_synonym`：缺少 fault diagnosis/detection、thermal runaway、BMS、PMSM、SHM 等同义词或缩写。
- `wrong_source`：检索源不适合主题，如工程材料题目过度依赖 PubMed。
- `off_topic_passed`：跑题文献通过筛选。
- `relevant_rejected`：相关文献被误拒。
- `metadata_noise`：标题/摘要/元数据不足导致误判。
- `translation_error`：中文工程概念翻译错误。
- `concept_dropped`：系统、材料、过程、方法、控制目标、性能指标、工况等核心概念丢失。
- `condition_mismatch`：运行工况、材料体系、设备类型或应用行业不匹配。
