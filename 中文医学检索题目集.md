# 中文医学文献检索题目集（广泛医学版）

用途：用于优化中文医学主题到英文学术检索计划、检索源选择、相关性筛选和文献分析流程。题目覆盖影像/AI、临床诊疗、药物治疗、公共卫生、护理康复、医学教育和卫生经济等多类医学检索场景。

标注位置：本文件只保存“检索题目”。每次实际检索后，候选文献会自动追加到 `检索标注记录.md`，请在那里填写 `人工判断`、`错误类型` 和 `备注`。

建议使用方式：

- 每次选 5-10 个题目跑检索，保存 `logs/search_audit_*.json`。
- 对返回文献人工标注为 `应收`、`待复核`、`应拒`。
- 把误收、漏收原因记录到“错误类型”列。
- 根据错误类型决定是补词表、改 LLM prompt、调筛选阈值，还是改数据源策略。

常用错误类型：

- `query_too_broad`：检索式过宽。
- `query_too_narrow`：检索式过窄。
- `missing_synonym`：缺少英文同义词、缩写或中文表达。
- `wrong_source`：检索源不适合主题。
- `off_topic_passed`：跑题文献通过筛选。
- `relevant_rejected`：相关文献被误拒。
- `metadata_noise`：元数据、标题、摘要质量差导致判断错误。
- `translation_error`：中文医学主题被错误翻译。
- `concept_dropped`：疾病、模态、任务、结局等核心概念被丢失。
