# 中文社科文献检索题目集（society 版）

用途：用于测试 `search_mode: society` 下的中文社科主题到英文学术检索式生成、检索源选择、相关性筛选和后续文献分析流程。题目覆盖社会学、教育学、劳动研究、公共政策、传播学、人口研究、城市研究、政治社会学、法社会学、社会心理与计算社会科学等场景。

推荐默认设置：
- `search_mode`: `society`
- 推荐检索源：Crossref、OpenAlex、Semantic Scholar、CNKI；计算社会科学、网络分析、计量经济方法可加 arXiv；公共卫生、心理健康、医学社会学可加 PubMed。
- 不建议把 PubMed 作为社科主题默认源，除非主题包含 public health、mental health、epidemiology、health policy、medical sociology 等明确健康社会科学方向。

标注位置：本文件只保存“检索题目”。每次实际检索后，候选文献会自动追加到 `检索标注记录.md`，请在那里填写 `人工判断`、`错误类型` 和 `备注`。

建议使用方式：
- 每次选 5-10 个题目跑检索，保留 `logs/search_audit_*.json`。
- 将返回文献人工标注为 `应收`、`待复核`、`应拒`。
- 重点记录误收、漏收、主题泛化、学科源选择错误、中文概念翻译错误。
- 根据错误类型决定补充社科词表、改 society prompt、调整筛选阈值，或调整检索源策略。

常用错误类型：

- `query_too_broad`：检索式过宽，泛化到大主题。
- `query_too_narrow`：检索式过窄，漏掉常见同义表达。
- `missing_synonym`：缺少英文同义词、理论名称、制度名称或中文表达。
- `wrong_source`：检索源不适合主题，例如纯社会政策题过度依赖 PubMed。
- `off_topic_passed`：跑题文献通过筛选。
- `relevant_rejected`：相关文献被误拒。
- `metadata_noise`：标题、摘要、来源元数据过少或噪声导致误判。
- `translation_error`：中文社科概念被错误翻译。
- `concept_dropped`：人口、制度、政策、地区、时期、方法、结果等核心概念丢失。
- `method_mismatch`：用户要求定性/民族志/因果识别/实验等方法，但返回方法不匹配。
- `context_mismatch`：地区、国家、平台、制度场景不匹配。
